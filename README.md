A simple author website.

Goals:
1. Provide the reader with a clear picture of the books I've written.
2. For each book, include a short summary, genres, and themes.
3. Where to read for free and how to buy each book (direct sales via PayPal).
4. Basic info about me.
5. How to contact me.

Initially, I used the AI helper in Visual Studio Code to assist me with the layout and create a basic draft of the site. I'm slowly changing this with code that I fully understand (I'm still learning). If the AI grabbed some code made by someone, I apologize; it wasn't my intention to use it without giving proper credit. Let me know if this is an issue so I can change it ASAP or credit you. Thanks for understanding.
